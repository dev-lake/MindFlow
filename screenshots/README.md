# 截图说明

请在此目录下添加以下截图：

## 需要的截图

1. **main-interface.png** - 主界面截图
   - 展示完整的应用界面
   - 包含会话列表、对话树、输入框等

2. **conversation-tree.png** - 对话树展示
   - 展示多分支的对话树结构
   - 突出显示节点之间的连接关系

3. **auto-layout.png** - 智能布局功能
   - 展示重新排列按钮
   - 对比排列前后的效果

4. **quote-feature.png** - 引用功能
   - 展示文本选择和引用按钮
   - 展示引用内容在输入框中的显示

## 截图要求

- 分辨率: 建议 1920x1080 或更高
- 格式: PNG
- 内容: 使用真实的对话内容，避免敏感信息
- 界面: 保持界面整洁，突出功能特点

## 如何添加截图

1. 使用截图工具捕获界面
2. 将图片保存到此目录
3. 确保文件名与 README.md 中引用的名称一致
